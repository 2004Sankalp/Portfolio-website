This website is a modern, responsive, and high-performance portfolio built to showcase my projects, skills, and professional journey.

## 🚀 Features

- **Modern UI/UX**: Built with a sleek, premium design using Tailwind CSS and Radix UI components.
- **Dynamic Animations**: Smooth transitions and interactive elements powered by Framer Motion.
- **Dark/Light Mode**: Fully functional theme switching with `next-themes`.
- **Responsive Design**: Optimized for all screen sizes, from mobile to ultra-wide monitors.
- **Contact Form**: Integrated contact section for professional inquiries.
- **Project Showcase**: Detailed cards for key projects with technology tags and descriptions.

## 🛠️ Tech Stack

- **Frontend**: React 18 with Vite
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Components**: Radix UI (Shadcn UI patterns)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Theming**: Next-Themes

## ⚙️ Local Development

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/2004Sankalp/Portfolio-website.git
   cd Portfolio-website
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🌐 Deployment

The project is configured for deployment via GitHub Pages.
